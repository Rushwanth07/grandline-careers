Conference Presentation

GrandLine Careers

AI-Powered Career Guidance for Youth Growth and Skill Development

A One Piece inspired platform for career discovery, skill mapping, and guided growth.

Slide 1
Problem

Why this project matters

  • Many students want growth but do not know which role fits them.
  • Career advice is often generic, fragmented, and not personalized.
  • Users struggle to connect current skills with future job opportunities.
  • Most platforms show information but do not give a clear next-step plan.
Core challenge: users do not need more content, they need clearer direction.
Slide 2
Solution

What GrandLine Careers delivers

  • Career path exploration through a guided, visual interface.
  • Role matching based on skills, interests, and goals.
  • AI-generated guidance with ranked roles and gap analysis.
  • A 12-week roadmap with practical milestones and weekly execution steps.
Value proposition: move users from confusion to clarity, and from interest to execution.
Slide 3
Audience

Target users

  • School and college students exploring careers
  • Freshers preparing for jobs or internships
  • Early professionals planning a transition
  • Learners seeking a structured improvement path

User questions we answer

  • Which career path fits my current skills?
  • What am I missing?
  • What should I do in the next 3 months?
  • Which resources should I prioritize?
Slide 4
Journey

User experience flow

  1. User lands on the homepage and explores career roles.
  2. User searches by interest or selects a crew specialty.
  3. User opens a detailed career page with roadmap phases.
  4. User clicks Begin Training.
  5. AI Crew Strategist generates personalized guidance.
  6. User receives strengths, ranked roles, skill gaps, roadmap, and weekly plan.
Slide 5
Features

Product features

Frontend

  • Interactive homepage and search

  • Google and mobile authentication

Backend

  • Career listing and search APIs
  • Career detail API
  • AI guidance API
  • Authenticated profile APIs
  • CORS and deploy configuration support
Slide 6
AI

How the AI Crew Strategist works

  • User enters current skills, interests, and goal.
  • Backend validates and normalizes the input.
  • Career roles are scored against the profile.
  • Top role matches are ranked.
  • Missing skills, quick wins, and resources are identified.
  • A 12-week roadmap and weekly coaching plan are generated.
Agent outputs: Profile Agent, Opportunity Agent, Gap Agent, Roadmap Agent, Coach Agent.

Slide 8
Architecture

System architecture overview

  • Frontend: React, Vite, Tailwind CSS, Framer Motion
  • Backend: Express.js REST API
  • Authentication: Firebase Auth
  • Profile handling: Firebase Admin and Firestore
  • Deployment: live frontend-backend integration with CORS allowlist
Request flow: UI sends user data to backend, backend validates and scores roles, UI renders structured guidance.
Slide 9
Insights

Key project insights

  • Youth need actionable guidance, not just information.
  • Role-fit scoring increases confidence by narrowing choices.
  • A themed interface improves engagement without reducing seriousness.
  • Structured roadmaps make AI output more practical than plain text.
  • Clean separation of APIs and strong validation improves reliability.
